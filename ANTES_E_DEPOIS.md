## 📱 Skip Link (Acessibilidade)

### ANTES
```
Usuário abre site → Vê logo da navbar e menu
                  → Pressiona Tab 20 vezes para chegar ao conteúdo
                  → Frustrante para teclado/screen reader
```

### DEPOIS
```
Usuário abre site → Pressiona Tab uma vez
                  → "Pular para conteúdo principal" aparece
                  → Pressiona Enter → Pula direto pro conteúdo
                  → ⭐ Economia de 19 Tabs!
```

### ANTES
```

→ Clica "Copiar código PIX"
→ alert() intrusivo aparece: "Código Pix copiado com sucesso!"
  [OK]
→ Usuário clica OK
→ :/ Experiência ruim (2000s style)
```

### DEPOIS
```
Usuário na seção "Doações"
→ Clica "Copiar código PIX"

  ┌──────────────────────┐

  └──────────────────────┘
```
